# אפליקציית SurfCheck Israel 🏄

ברוכים הבאים ל-**SurfCheck**, אפליקציית חובה לכל גולש בישראל.
האפליקציה מרכזת נתוני ים ומזג אוויר בזמן אמת לחופי הים התיכון, מנהריה ועד אשקלון.

### 👇 רוצים לראות את האפליקציה עובדת? 👇
האפליקציה נבנתה ב-React Native וניתן להריץ אותה ישירות בדפדפן:
https://snack.expo.dev/embedded/@zamirh66/tahazit-galim?preview=true&platform=web&theme=light
---

## ✨ פיצ'רים מרכזיים
* **📍 רשימת חופים ארצית:** מעבר מהיר בין חופים נבחרים (נהריה, עכו, נתניה, הרצליה, ת"א, בת ים, ראשל"צ, אשדוד, אשקלון).
* **🌊 נתוני זמן אמת:** גובה גלים, כיוון רוח, עוצמת רוח וטמפרטורה.
* **🧠 המלצות חכמות:** אלגוריתם הממליץ האם להיכנס למים בהתאם לגובה הגל.
* **📅 תחזית שבועית:** סינכרון מלא של תחזית גלים ומזג אוויר ל-7 הימים הקרובים.
* **📱 מותאם למובייל:** עיצוב רספונסיבי שמתאים לאייפון ואנדרואיד (כולל תמיכה ב-Notch/Safe Area).

## 🛠️ טכנולוגיות
* **React Native & Expo**
* **Open-Meteo API** (משיכת נתונים משני מקורות והצלבת מידע)
* **JavaScript (ES6)**
